Report a Bug

The PostgreSQL community takes pride in releasing software that will reliably store your data. If you believe you have discovered a bug, please follow the button below and follow the instructions on how to submit a bug report.

Submit a Bug Report

If you believe you have found a security issue, please read the section below.

Report a Security Issue

If you believe you have found a security issue, please send an email to security@postgresql.org.

Support

If you have general questions about PostgreSQL or are looking for help with using PostgreSQL, please visit the support section of the website to find out how to get in touch.

Code of Conduct Committee

If you are the recipient of behavior that violates the PostgreSQL Code of Conduct, please report it promptly to the Code of Conduct Committee at coc@postgresql.org. In the unfortunate event that you wish to make a complaint against some member of the Committee, you may instead contact any of the other Committee member(s) individually.

Fundraising and donations

For information about fundraising and donations, including financial sponsorship for PostgreSQL-related activities, please visit the donation page or contact the funds group directly at funds-group@postgresql.org.

User Group Liaison

If you are looking for support for a PostgreSQL User Group, looking to start a user group, or need resources for a user group, please contact usergroups@postgresql.org. For a list of active PostgreSQL User Groups, please visit the local user groups page.

Press Enquiries

For press enquiries, please refer to the Press section.

Webmasters

Contact the PostgreSQL webmasters at webmaster@postgresql.org.
