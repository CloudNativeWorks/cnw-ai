FAQ

A MongoDB Enterprise Advanced subscription includes a commercial license for MongoDB Enterprise Server. MongoDB Enterprise Server provides several capabilities unavailable in the MongoDB Community Edition. This includes in-memory storage engine for self-managed deployments, auditing self-managed deployments, and Kerberos authentication on self-managed deployments.

Core server capabilities are broadly comparative across the two editions. However, a MongoDB Enterprise Advanced subscription includes additional operational and management tools, a commercial license, and access to 24/7/365 technical support.

MongoDB Atlas and MongoDB Enterprise Advanced are two great ways of running the popular modern database. Ultimately it comes down to deployment location and requirements.

MongoDB Atlas is available in 125+ regions across Amazon Web Services (AWS), Google Cloud, and Microsoft Azure.

MongoDB Enterprise Advanced is ideal if on-premises deployment, or in a private or hybrid cloud is a key requirement.
